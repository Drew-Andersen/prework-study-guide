HTML

  • The head element contains information about the webpage.
  • The body element represents the visable content shown to the user.
  • The head element is for the browser and is not seen by the user.

CSS

  • A margin indicates how much space we want around the outside of the element.
  • A padding indicates how much space we want around the content inside an element.
  • A box-shadow indicates a "shadow" around the element.
  • Color can be indicated several ways: A hex # or the color name (gray).
  • Width will indicate how wide we want the element to be.
  • Height will indicate how tall we want to element to be.

GIT

  • git status: checks what branch we are currently in
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds the modifications to the staging area
  • git commit -m commit-mesage: commits to GitHub
  • git pull origin main: pulls the main branch from GitHub repo
  • git push origin branch-name: pushes the feature branch to GitHub
  • git checkout main: switches to the main branch
      • How to push your code to GitHub
    1. Navigate to the correct directory in the terminal
    2. Use "git add -A"
    3. Then git commit -m "your message here"
    4. Then git pull origin main
    5. Then git pus origin "your feature name here"
    6. Next to GitHub and review/merge the branch to the main

Javascript

  • A variableis a names container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function holds code so that it can easily be called again in the code.
  • A for loop "loops" through the code until it reaches the designated end.
